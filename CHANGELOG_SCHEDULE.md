# Улучшения системы генерации расписания

## Дата: 14 апреля 2026 г.

### Описание изменений
Система расписания теперь генерирует расписание в красивом табличном формате как на эталонной картинке с правильной структурой и оформлением.

## Основные изменения

**Файл:** `app/templates/index.html`

**Что изменилось:**
- Добавлено явное требование генерировать расписание в markdown таблице
- Указан правильный формат с заголовком (Время, 1 класс, 2 класс, 3 класс, 4 класс)
- Добавлены требования к времени уроков и структуре (Понедельник, Вторник)
- Обновлен system prompt для лучших результатов

**Пример формата вывода:**
```

| Время | 1 класс | 2 класс | 3 класс | 4 класс |
|-------|---------|---------|---------|---------|

| 9:30-10:15 | Русский язык | Математика | Русский язык | Математика |
```

После этих изменений расписание генерируется и отображается следующим образом:

1. **На фронте (браузер):**

## Как использовать

1. Запустите сервер:
```bash
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

2. Откройте браузер на `http://localhost:8000`

3. Заполните форму:
   - **Основной промпт**: используется обновленный prompt по умолчанию (требует markdown таблицы)
   - **Системный промпт**: специализирован на создание школьного расписания
   - **Провайдер ИИ**: выберите нужного (YandexGPT, GigaChat, OpenAI и т.д.)

4. Нажмите "Сгенерировать расписание"

5. Результат отобразится как красивая таблица

6. Нажмите "Скачать Excel" для экспорта в Excel файл

## Технические детали

### Требования к формату расписания от AI

Система требует от AI генерировать расписание в следующем формате:

```
<День недели>

| Время | 1 класс | 2 класс | 3 класс | 4 класс |
|-------|---------|---------|---------|---------|
| <время урока> | <предмет> | <предмет> | <предмет> | <предмет> |
...
```

### Регулярные выражения для парсинга

- **Время:** `^\s*\d{1,2}:\d{2}\s*-\s*\d{1,2}:\d{2}\s*$`
- **Класс:** `(\d+)\s*класс` (регулярное выражение, игнорирующее регистр)
- **День недели:** прямое совпадение с одним из: Понедельник, Вторник, Среда, Четверг, Пятница, Суббота, Воскресенье
