## Или используйте промпт по умолчанию

Нажмите кнопку "↻ Сброс" чтобы вернуть стандартный промпт, который требует генерирование расписания на полную неделю в markdown таблице формате с правильной структурой.

## Требования к формату для правильного парсинга:

1. **Каждый день начинается с названия**: "Понедельник", "Вторник", "Среда", "Четверг", "Пятница" на отдельной строке
2. **Пустая строка** после названия дня
3. **Markdown таблица** с вертикальными разделителями `|`:
   - Первая строка: Время | 1 класс | 2 класс | 3 класс | 4 класс
   - Вторая строка: разделители `|-------|---------|---------|---------|---------|`
   - Строки данных: время и предметы для каждого класса
4. **Формат времени**: `ч:мм-ч:мм` (например: 8:30-9:15, 9:30-10:15)
5. **Пустая строка** между днями недели (опционально, но приветствуется)

## Пример неправильного формата (НЕ будет распарсен):

```
Понедельник - Вторник
8:30-9:15  Математика  Русский язык  Математика  Русский язык
9:30-10:15 Русский язык Математика Русский язык Математика
```

## Ключевые предметы для использования:

- Математика
- Русский язык
- Литературное чтение
- Окружающий мир
- Английский язык
- Физкультура
- Музыка
- Изо (Изобразительное искусство)
- Труд / Технология
