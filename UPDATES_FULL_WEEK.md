# ✅ Обновление: Расписание теперь генерируется на полную неделю (Пн-Пт)

## 📋 Что изменилось

### 1. **DEFAULT_PROMPT обновлен**
- **Было**: Расписание только на понедельник и вторник
- **Стало**: Полная неделя (Пн-Вт-Среда-Чт-Пт)

### 2. **Требования к содержанию расширены**
- Теперь в prompt явно указано требовать расписание на 5 дней в неделю
- Добавлена рекомендация обеспечить разнообразие расписания от дня к дню
- System prompt обновлен для работы с полной неделей

### 3. **Пример формата включает полную неделю**
- Показан формат для Пн-Вт-Среда-Чт-Пт
- Каждый день отформатирован в правильную markdown таблицу

## 🚀 Как использовать

1. **Откройте приложение**: http://localhost:8000
2. **Нажмите "↻ Сброс"** чтобы загрузить обновленный prompt
3. **Нажмите "🚀 Сгенерировать расписание"** 
4. **AI сгенерирует расписание на всю неделю**
5. **Результат отобразится в красивой таблице с разделением по дням**
6. **Нажмите "⬇️ Скачать Excel"** для экспорта

## 📊 Структура выхода

Расписание генерируется в следующем формате:

```
Понедельник

| Время | 1 класс | 2 класс | 3 класс | 4 класс |
|-------|---------|---------|---------|---------|
| 8:30-9:15 | Предмет | Предмет | Предмет | Предмет |
...

Вторник

| Время | 1 класс | 2 класс | 3 класс | 4 класс |
|-------|---------|---------|---------|---------|
...

Среда
...

Четверг
...

Пятница
...
```

## ✨ Особенности

- ✅ Полная неделя (5 дней)
- ✅ 4 класса (1-4)
- ✅ 4 или 5 уроков в день (зависит от класса)
- ✅ Разнообразное расписание день за днем
- ✅ Правильно распределенные спецпредметы
- ✅ Красивая таблица с фиолетовым градиентом
- ✅ Excel экспорт с полной неделей

## 📝 Команды в браузере

- 🚀 **Сгенерировать расписание** - генерирует расписание по обновленному prompt на полную неделю
- ⬇️ **Скачать Excel** - скачивает файл с расписанием в Excel формате
- 📋 **Таблица / Исходный текст** - переключение между видами отображения
- ↻ **Сброс** - очищает форму и загружает стандартные значения

## 🔄 Что происходит на бэкенде

1. Ваш prompt отправляется к выбранному AI провайдеру
2. AI генерирует расписание в markdown таблице формате на полную неделю
3. Фронтенд парсит markdown таблицы
4. Расписание группируется по дням
5. Отображается в красивой HTML таблице
6. При скачивании Excel - создается структурированный файл с каждым днем в отдельной таблице
